# Actively Maintained Minecraft Projects
## Server Softwares
- Jade
> A multithreaded and performant server software for large open-world multiplayer servers.
- Opal
> A paper fork with better data management and development capabilities.
## Plugins
- Fungus
> A private anti-cheat for modern servers.
- WhirlChat
> An all-in-one channel-based chat plugin for formatting & filtering.
- Mannequins
> A combat/danger logout prevention plugin, using mannequin entities.
- NetworkLink
> A linking plugin to sync players and data across multiple game servers and proxies.
## Mods
- Juggle
> A script-based cheat/utility mod. Primarily used for testing Fungus.
- APILookup
> An in-game user interface for hitting APIs and parsing JSON responses.
